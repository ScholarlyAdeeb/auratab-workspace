# AuraTab Workspace 🌌

> **Organize your learning tracks, eliminate tab fatigue, and maintain deep-focus developer flow.**

A premium, ultra-minimalist browser extension that replaces the default New Tab page with a high-end, distraction-free developer dashboard. 

---

## 💡 The Inspiration
Hi, I'm Adeeb Khan, an Electrical Engineering student at DTU. I built this minimalist workspace to solve a personal frustration: losing focus while jumping between multiple course tracks. It cleanly organizes and tracks learning playlists in one central dashboard, removing the friction and distractions of constant searching.

---

## ✨ Features

- **Chrono-Aesthetic Engine:** Context-aware, non-cliché dynamic greetings that cycle randomly through 12 distinct action-focused phrases based on your local system time.
- **Dynamic Multi-Playlist Sidebar:** Drop in your YouTube study playlist links. AuraTab automatically scrapes total video counts through a no-auth proxy, caching your progression circles cleanly in `localStorage` with left/right-click quick increments.
- **Smarter Command Palette:** Press `/` to reveal a blur-glass terminal interface to search shortcuts, switch tracking targets, or execute system commands.
- **Zen Ambient Screen Saver:** Type `/zen` into the command palette to fade the workspace components away. The entire screen scales into a micro-dimmed, textless analog clock face to serve as a resource-efficient screensaver while you work.
- **Hardware-Accelerated Themes:** Switch between premium Dark and Light viewports alongside 4 custom radial aura glows (Deep Cosmic, Matrix Neon, Sunset Crimson, Slate Studio).

## 🚀 Installation & Setup (100% Free)

Because this is an open-source utility, you can bypass the Chrome Web Store paywalls and run it natively as an unpackaged developer module:

### 1. Clone or Download the Artifact
- Click the green **Code** button at the top right of this repository and select **Download ZIP**.
- Extract the archive file to a folder on your laptop (e.g., `C:\Users\Documents\AuraTab`).

### 2. Turn on Developer Mode
- **Google Chrome:** Open a new tab and navigate to `chrome://extensions/`
- **Microsoft Edge:** Open a new tab and navigate to `edge://extensions/`
- Locate the **Developer Mode** toggle switch in the browser pane and flip it **ON**.

### 3. Mount the Extension Folder
- Click the **Load unpacked** button in the top-left menu.
- Select the folder containing your `manifest.json`, `newtab.html`, and `script.js` files.
- Open a brand-new tab window to initialize your clean command center!
